# Event Sourcing

The eventsourcing package provides a simple set of abstractions for working with event sourcing.
Like event store and aggregate utilities.

[https://jsr.io/@nimbus/eventsourcing](https://jsr.io/@nimbus/eventsourcing)

### Deno

```bash
deno add jsr:@nimbus/eventsourcing
```

### NPM

```bash
npx jsr add @nimbus/eventsourcing
```

### Bun

```bash
bunx jsr add @nimbus/eventsourcing
```
